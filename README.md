# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover
sales trends, customer behavior, and product performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Analysis
- Data cleaning and preprocessing
- Descriptive statistics
- Time series analysis of monthly sales
- Customer and product category analysis
- Data visualizations (line, bar, pie charts)

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset (`retail_sales_dataset.csv`)
3. Run all cells sequentially
